Punto 5

Var vs Let vs Const

Es crucial entender el alcance (scope) de las variables para evitar errores en aplicaciones grandes.

var

Tiene alcance de función. Permite redeclaración. Sufre de hoisting (se eleva al inicio).

let

Alcance de bloque. Ideal para variables que cambiarán su valor. No permite redeclarar en el mismo bloque.

const

Alcance de bloque. No permite reasignar el valor. Obligatorio inicializarla al declararla.

let puntaje = 10;
puntaje = 20; // Correcto

const pi = 3.1416;
// pi = 3.15; // Error: Assignment to constant variable
Punto 6

Funciones de Flecha (Arrow Functions)

Ofrecen una sintaxis más limpia y resuelven problemas del contexto de this.

// Forma tradicional
const suma = function(a, b) { return a + b; };

// Arrow function
const sumaFlecha = (a, b) => a + b;

// Si solo tiene un parámetro, no necesita paréntesis
const duplicar = n => n * 2;
Punto 7

Template Strings

Permiten concatenar texto y variables de forma legible usando el acento invertido (`).

const usuario = "Juan";
const rol = "Admin";

// Interpolación con ${}
const saludo = `Bienvenido ${usuario}, tu rol es: ${rol}`;
console.log(saludo);
Punto 8 y 9

Objetos y Destructuring

La desestructuración permite extraer propiedades directamente a variables.

const producto = {
    nombre: "Laptop",
    precio: 1500,
    stock: true,
    detalles: { color: "Gris" }
};

// Destructuring básico
const { nombre, precio } = producto;

// Destructuring de objetos anidados
const { detalles: { color } } = producto;

console.log(nombre, color); // Laptop, Gris

Practicando con Objetos

Combinación de métodos, objetos literales y manipulación.

const curso = {
    titulo: "Programación II",
    tecnologias: ["JS", "HTML", "CSS"],
    mostrarInfo() {
        console.log(`Curso: ${this.titulo}`);
    }
};

// Congelar objeto (no permite cambios)
Object.freeze(curso); 
// curso.titulo = "Nuevo"; // No funcionará

// Obtener llaves o valores
console.log(Object.keys(curso)); 
console.log(Object.values(curso));
Punto 13 y 14

Métodos Map y Filter

Estos métodos son esenciales porque **no mutan** el array original.

.map()

Crea un nuevo array transformando cada elemento.

const num = [1, 2, 3];
const dobles = num.map(n => n * 2); // [2, 4, 6]

.filter()

Crea un nuevo array con elementos que cumplan una condición.

const notas = [100, 50, 80];
const aprobados = notas.filter(n => n >= 60); // [100, 80]

Métodos del Objeto DOM

Basado en la estructura de gestión de nodos y atributos:

getAttribute(atrib): Retorna el valor del atributo del objeto indicado.
setAttribute(atrib, val): Modifica o asigna el nuevo valor al atributo del objeto indicado.
removeAttribute(atrib): Elimina el atributo del objeto indicado.
document.createElement(tag): Crea y retorna un objeto HTML indicado en el parámetro.
objeto.appendChild(hijo): Añade un objeto al final de la lista de "hijos" del objeto. 
// 1. Crear un párrafo
const parrafo = document.createElement('p');

// 2. Asignar atributos y texto
parrafo.setAttribute('id', 'info-extra');
parrafo.textContent = "Este es un texto dinámico";

// 3. Inserción
document.body.appendChild(parrafo);

// 4. Lectura y eliminación
console.log(parrafo.getAttribute('id')); // "info-extra"
parrafo.removeAttribute('id');

Diccionario de Eventos en JavaScript

El método addEventListener(): Es el estándar de HTML5. Recibe el nombre del evento (sin el "on"), la función a ejecutar y un booleano para la propagación.

onBlur: Al deseleccionar campo. Input/Textarea
onChange: Al cambiar contenido. Select/Input
onFocus: Al activar campo. Input
onLoad: Al cargar documento. Body
onMouseOver: Mouse encima. Universal
onMouseOut: Mouse alejándose. Universal
onSelect: Al seleccionar texto. Input
onSubmit: Al enviar formulario. Form
onClick: Clic simple. Button/a
onDblClick: Doble clic. Button
onKeyDown: Al pulsar tecla. Input/Global
onMouseDown: Al pulsar botón mouse. Universal 
const miBoton = document.querySelector('#btn');

// El nombre del manejador para el evento 'click' es 'onclick'
miBoton.addEventListener('click', (e) => {
    console.log("Click detectado");
});
Punto 11 y 12

Fetch API y Async / Await

Permite realizar peticiones asíncronas a servidores externos sin bloquear el hilo principal de ejecución.

// Función asíncrona
const obtenerClima = async () => {
    try {
        // Fetch devuelve una promesa
        const respuesta = await fetch('https://api.ejemplo.com/clima');
        
        // Validamos la respuesta
        if (!respuesta.ok) throw new Error("Fallo en la petición");
        
        // Esperamos la conversión a JSON
        const datos = await respuesta.json();
        console.log(datos);

    } catch (error) {
        console.error("Error:", error.message);
    }
};

obtenerClima();
